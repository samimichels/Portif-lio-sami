Desafio Sudoku: Um Jogo de Lógica

Este projeto é um jogo de Sudoku feito para ajudar a treinar o raciocínio lógico. O usuário pode preencher os campos com números de 1 a 9, seguindo as regras tradicionais do jogo: cada número deve aparecer apenas uma vez em cada linha, coluna e quadrado.

Foi desenvolvido com HTML para a estrutura, CSS para o estilo e JavaScript para toda a lógica de verificação e interação. Esse jogo me ajudou muito a entender estruturas de repetição e condições no JavaScript.
